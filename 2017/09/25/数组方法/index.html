数组方法：

push()返回的是改变后的数组长度，pop（）返回被移除的元素
1、concat()：会先创建当前数组的一个副本，然后将接收到的参数添加到这个副本的末尾，最后返回新构建的数组。
2、slice（）和substring（）都可以返回字符串指定区间
slice() :基于当前数组中的一个或多个项创建一个新数组，可以接受一或两个参数，即要返回项的起始和结束位置。在只有一个参数的情况下，slice（）方法返回从该参数指定位置开始到当前数组末尾的所有项。如果两个参数，该方法返回起始和结束位置之间的项，但不包括结束位置的项。slice（）不会影响原始数组。
如果slice（）方法的参数中有一个负数，则用数组长度加上该数来确定相应的位置。例如，在一个包含5项的数组上调用slice（-2，-1）与调用slice（3,4）得到结果相同。如果结束位置小于起始位置，返回空数组
区别：对负数参数的处理方式，substring（）会将它视为从0开始的计数形式，而slice则会将它与字符串长度相加。因此，如果我们传给他们的参数是（1，-1）的话，实际情况分别是substring（1,0）和slice（1,s.length-1）

3、splice（）始终返回一个数组，数组包含从原始数组中删除的项，如果没有删除返回一个空数组。
删除：指定2个参数，要删除的第一项的位置和要删除的项数，如splice（0,2）会删除前两项。
插入：指定3个参数：起始位置，0（要删除的项数），要插入的项

4.位置方法：
（1）indexOf（）接收一或两个参数：要查找的项和查找起始点位置索引。
（2）lastIndexOf（）末尾向前查找
5、迭代方法
（1）every（）：对数组中的每一项运行给定函数，如果该函数对每一项都返回true，则返回true
（2）filter（）：对数组中的每一项运行给定函数，返回该函数会返回true的项组成的数组。
（3）forEach（）：对数组中的每一项运行给定函数，没有返回值
（4）map（）：对数组中的每一项运行给定函数，返回每次函数调用的结果组成的数组。
（5）some（）：对数组中的每一项运行给定函数，如果该函数对任一项返回true，则返回true
6迭代方法：
reduce（）和reduceRight（）。这两个方法都会迭代数组的所有项，构建一个最终返回的值。
其中reduce（）从数组第一项开始，逐个遍历数组。reduceRight（）则从最后一项开始。
接收4个参数：前一个值，当前值，项的索引和数组对象。这个函数返回的任何值都会作为第一个参数传给下一项。
使用reduce（）求数组所有值得和：
var values = [1,2,3,43,3]
var sum = values.reduce(function(prev,cur,index,array) {return prev+cur; })
alert(sum)

正则表达式为参数的字符串方法：
match（）：返回包含匹配内容数组
search（）：返回第一个匹配内容所在位置
replace（）：将匹配的文本替换成指定的字符串
split（）：根据指定的正则表达式将目标字符串分割成若干个数组元素
